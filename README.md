# Orquestación de Equipos de Agentes LLM con Roles Especializados para Resolución Colaborativa de Tareas Complejas de Ingeniería de Software

**Stack tecnológico:** Python · LangGraph · LangChain

**Fecha de inicio:** 2026-04-22

## Estructura del proyecto

```
TFG_MultiAgente/
├── doc/                  Documentación escrita del TFG
│   ├── capitulos/        Capítulos de la memoria en Markdown
│   └── referencias/      Bibliografía en formato BibTeX
├── src/                  Código fuente del sistema multi-agente
│   ├── agents/           Definición e implementación de cada agente especializado
│   ├── graph/            Grafos LangGraph que orquestan los flujos de agentes
│   ├── tools/            Herramientas externas invocables por los agentes
│   ├── state/            Esquemas de estado compartido entre agentes
│   └── evaluation/       Métricas y scripts de evaluación automática
├── experiments/          Configuraciones y resultados de experimentos
├── figures/              Figuras y diagramas para la memoria
└── README.md             Este archivo
```
